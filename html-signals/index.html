Svenska signalbilder i HTML

Ovan visas alla korrekta signalbilder som svenska huvudljussignaler kan visa. Dessa genereras med JavaScript utifrån en array med värden. Till exempel skrivs signalen "Kör, vänta stopp" (5:e fr. v) såhär:

let signal = ["g", "0", "b", "0"];

// 0 = släckt, g = grön, r = röd
// b = grön blink, w = vit blink

Det roliga med det är att man kan göra signaler som verkligen inte finns.

let fake = ["w", "b", "r"];
